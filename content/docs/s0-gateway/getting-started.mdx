# Getting Started with S0

This guide will help you get up and running with the S0 IoT Gateway quickly.

## What You'll Need

### Hardware

- S0 IoT Gateway module
- USB-C cable for power and programming
- Optional: S0 Base Board for expanded functionality
- Optional: BeagleV-Fire board for HAT mode

### Software

- [Zephyr SDK](https://docs.zephyrproject.org/latest/develop/getting_started/index.html)
- [ESP-IDF](https://docs.espressif.com/projects/esp-idf/en/latest/esp32c6/get-started/) (for ESP32-C6)
- Serial terminal (minicom, screen, or PuTTY)
- Git

### Accounts

- [Magistrala](https://github.com/absmach/magistrala) instance (cloud or local)
- Optional: NB-IoT/LTE-M SIM card with data plan
- Optional: OCI registry for Wasm modules
